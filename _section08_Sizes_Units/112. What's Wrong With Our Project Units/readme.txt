Neste modulo da AUla 120 estaremos estudando max-WIDTH e min-WIDTH


MAX-WIDTH: PX : Faz com que uma DIV que tenha uma IMG definada com PÍXEL, tenha um tamanho MAXIMO, com 
isto n temos aquele efeito de imagem alongada:

MIN-WIDTH: PX : Faz com que uma DIV que tenha uma IMG definada com PÍXEL, tenha um tamanho MÍNIMO, com 
isto n temos aquele efeito de imagem muito retuzida:

Usaremos NO CUSTOMERS.CSS os Exemplos das classes: 
<div class="testimonial__div__image__left"> na linha 20 do CSS
    width: 65%;
    max-width: 580px;
    min-width: 220px;

<div class="testimonial" id="customer__img-2"> na linha 30 do CSS
    width: 65%;
    max-width: 580px;
    min-width: 220px;
